Random.js- generate random data

Generate A Random String

random.string(length) generates a random random string of length length
NOTE*:This string can contain any character

createRandomString(string,length) generates a random random string of length length and contains characters within string

Generate A Random Integer

random.int.range(min,max) generates a random integer in range min-max both inclusive

random.int.length(length) generates a random integer of length length

Generate A Random Float

random.float(min,max) generates a random float in range min-max

Generate A Random E-Mail

random.email(length) generates a random email with a username of length length

Generate A Random URL

random.url(length) generates a random url with length length
NOTE*:length parameter of url must be greater than 8

Generate A Random Date

random.date(a,b) generates a random date between year a and year b

If a and b are unspecified a is considered as 0 and b is considered as the current year

Get A Random Data From A Array

random.array.getData(array)returns a random data from the array array
